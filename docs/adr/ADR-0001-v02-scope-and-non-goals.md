# ADR-0001: v0.2 Scope and Non-Goals

## Status

Accepted for MVP implementation.

## Decision

SmartSleep Alarm v0.2 validates the Apple Watch smart alarm chain. It does not implement sleep reports, sleep stage detection, cloud sync, audio recording, medical claims, ML personalization, subscriptions, Android, or complex task alarms.

## Consequences

Engineering work prioritizes reliability, local diagnostics, device spikes, and a narrow vertical slice over feature breadth.
